# 02 · MDP 与网格世界：状态、动作、奖励、未来

Bandit 没有“未来”。现在我们加入未来：**你走一步会改变你接下来看到的世界**。

这类问题通常用 **MDP（马尔可夫决策过程）**建模。

## MDP 五件套（不怕，只有名词）
- 状态集合 S
- 动作集合 A
- 转移概率 P(s'|s,a)：在 s 做 a，下一步到 s' 的概率
- 奖励函数 R(s,a,s') 或 R(s,a)
- 折扣因子 γ

“马尔可夫”意思：
> 未来只和**当前状态**有关，不需要完整历史。

## 网格世界（Gridworld）
我们用一个 2D 地图：
- 你是一个点
- 可以上下左右移动
- 到达终点 +1
- 撞墙不动 -0.1（或 0）

这个例子好处：
- 你能直观看到“短期吃亏，长期更赚”的情况
- 非常适合理解价值函数和动态规划

## 为什么要折扣 γ？
直觉上有三种理由：
1. “现在的 1 块钱”比“以后可能的 1 块钱”更确定
2. 数学上更稳定（避免无限回报发散）
3. 在持续任务里，折扣让“近期”更重要

## 两类价值函数
- **状态价值**：V(s) = 从 s 开始，按策略 π 行动的期望累计折扣奖励
- **动作价值**：Q(s,a) = 从 s 开始先做 a，再按 π 行动的期望累计折扣奖励

## Bellman 直觉（不用背公式）
价值函数有一个非常关键的递推关系：
> “现在的价值 = 现在拿到的奖励 + 折扣后的下一步价值”

这就是后面 Q-learning/DQN/PPO 的“底层逻辑”。

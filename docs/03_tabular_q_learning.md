# 03 · 表格型 Q-learning：最经典的 RL 入门算法

> 这一章的目标：你要能清晰说出 **Q-learning 在做什么、为什么需要探索、更新式每一项是什么意思**。

---

## 0. TD 更新在干什么（把 Q 往 target 拉）

```mermaid
flowchart LR
  Qold[当前估计 Q(s,a)] -->|比较| delta[TD error δ = target - Q(s,a)]
  target[target = r + γ·max Q(s',·)] -->|比较| delta
  delta -->|用学习率 α 更新| Qnew[Q(s,a) ← Q(s,a) + α·δ]
```

## 1. 从“背公式”换成“修正打分”
你可以把 Q(s,a) 当作：
- 你对“在 s 做 a 之后长期能拿多少分”的**当前估计**

Q-learning 就是一种不断修正这个估计的方法。

你每经历一次 `(s, a, r, s')`，就获得了一个新的证据：
- 现在得到 r
- 下一步到了 s'
- 未来在 s' 的最好前景大概是 `max_a' Q(s',a')`

于是你会说：
> “我之前对 Q(s,a) 的估计，应该往 `r + γ * max Q(s',·)` 靠近。”

---

## 2. 一条更新式（你需要理解每个符号）
常见写法：

\[ Q(s,a) \leftarrow Q(s,a) + \alpha \Big( r + \gamma \max_{a'} Q(s',a') - Q(s,a) \Big) \]

逐项解释：
- **α（学习率）**：你相信新证据的程度（越大改得越猛）
- **r**：你刚拿到的即时奖励
- **γ max Q(s',·)**：你对未来最乐观的估计
- **括号里**是 TD error（时序差分误差）：
  - target（目标） - current（当前估计）

直觉：
- 如果 TD error > 0：说明你低估了，Q 要上调
- 如果 TD error < 0：说明你高估了，Q 要下调

---

## 3. 为什么要 `max`？（Q-learning 的“离策略”特征）
Q-learning 使用 `max_a' Q(s',a')`，意味着：
- 不管你实际下一步是否会探索
- 更新时都假设“未来会采取最优动作”

这使得 Q-learning 是 **Off-policy（离策略）**：
- 行为策略可以是 ε-greedy（带探索）
- 但学习目标是最优贪心策略

对比：SARSA 是 On-policy（用实际选到的 a' 更新）。

---

## 4. 探索 vs 利用：ε-greedy 为什么够用
训练时常用 ε-greedy：
- 大概率选 Q 最大动作
- 小概率随机

为什么要随机？
- 如果你永远不尝试某个动作，你永远不知道它是否更好

工程经验：
- ε 可以从大到小衰减
- 也可以保持一个小下限（适应非平稳）

---

## 5. Q-learning 在 Gridworld 里会学到什么
在网格世界中：
- 早期 Q 全是 0 → 乱走
- 只要偶然到达终点，终点附近的 Q 会先被抬高
- 这个高值会一格一格往起点方向“反传”（因为 Bellman 递推）

所以你会看到：
- 回合平均步数变少
- 回合总回报变大

---

## 6. 常见坑（新手必看）
1) **奖励太稀疏**：很难偶然摸到终点 → 学不起来（要改奖励/加 shaping）
2) **学习率太大**：震荡；太小：学得慢
3) **γ 太小**：看不远，只追逐近利
4) **探索不足**：早早锁死次优路径

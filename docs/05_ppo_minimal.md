# 05 · PPO：现代主力 RL 算法之一（直觉版）

DQN 适合离散动作，但对连续动作、复杂策略往往不如“策略梯度”路线自然。
**PPO（Proximal Policy Optimization）**是目前工业界/研究里最常用的通用 RL 算法之一。

## 从“直接学策略”开始
我们让一个神经网络输出策略：π(a|s)。
- 做了动作 a
- 拿到回报（或优势 advantage）
- 如果这个动作带来更高回报，就让网络以后更容易选到它

这就是策略梯度的直觉。

## 为什么需要 PPO 的“限制更新”
直接用策略梯度会出现：
- 每次更新把策略改得太猛 → 训练崩

PPO 的关键想法：
> **每次更新别走太远**。

实现方式（直觉）：
- 计算“新策略/旧策略”的概率比值 r
- 如果 r 偏离 1 太多，就把目标函数裁剪（clip）

这样可以大幅提高稳定性。
